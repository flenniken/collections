You view my best photos with this app. Photos are grouped into collections where each collection is a trip, photo shoot, or related topic. You can read about each photo and collection with the associated descriptions.

To learn how to use the app read the user guide (readme) on github. Developers can learn how to build and modify the source there too.

You install the icon by touching the share icon at the bottom center, scrolling down and tapping "Add to Home Screen".
